# 📝 Community Posts App

A simple Community Posts web application built using **Node.js**, **Express.js**, **EJS**, and **RESTful APIs**. Users can create, view, edit, and delete posts, demonstrating complete CRUD operations.

---

## 🚀 Features

- ➕ Create a new post
- 📖 View all posts
- ✏️ Edit an existing post
- 🗑️ Delete a post
- 🌐 RESTful Routing
- 🎨 Simple and responsive UI using HTML & CSS

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- EJS
- HTML5
- CSS3
- JavaScript
- REST API
